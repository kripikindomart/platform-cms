## 🚀 QUICK START

### Prerequisites
- Node.js 18+
- PostgreSQL
- Redis

### Installation

```bash
# Clone repository
git clone <repository-url>
cd platform-cms

# Backend setup
cd backend
npm install
cp .env.example .env
# Configure .env with database credentials
npm run db:push
npm run dev

# Frontend setup (new terminal)
cd frontend
npm install
npm run dev
```

### Access
- **Frontend:** http://localhost:3000
- **Backend API:** http://localhost:3001
- **Template Gallery:** http://localhost:3000/dashboard
