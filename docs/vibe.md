Ide aplikasi saya:

Platform aplikasi starter yang akan menjadi template untuk aplikasi aplikasi yang lain
aplikasi ini harus memeperhatikan beberapa module dasar atau core yang mudah di kembangkan oleh programmer lain atau junior programmer dan model ai yang lebih murah
module core ini terdiri dari beberap aspek yang memperhatikan performa aplikasi dan monitoring serta ada bbeberapa fitur memudahkan user 
maintenance dan builder, khusus builder ini sediakan builder cli dan ui 
untuk builder cli ini untuk mempermudah AI lain atau programmer lain membuat module atau crud secara langsung 
untu builder kita buat khusus ui agar memudahkan saya untuk membuat module lain secara ui 
untuk tahap awal buat kerangka awal untuk core nya mulai dari arsitektur awal yang proper seperti framework terkenal lainnya 
lalu setelah moodul core selesai buat module cli nya untuk kamu gunakan agar memudahkan kamu genrate module lain agra sesuai template 
perthatikan aspek aspek berikut , scurity harus diperhatikan input output harus di sanitasi untuk upload pun sama perhatikan celah keamanan mendetail
kesesuaian response data ketika diload harus seragam, pesan erro feedback tiap input output harus jelas terutama jika ada error harus dijabarkan secara mendetail
bukan hanya dibungkus keseluruhan tapi satu item untuk satu error, jangan pernah menggunakan emoji di tiap ui nya kecuali di cli, pastikan desain clean dan bagus
boleh menggunakan shadcnui untuk database kita menggunakan db drive agar bisa di pindah pindah database untuk saat ini kita gunakan postgresql tapi sediakan db drive 
untuk menggunakan db lain misal mysql gunakan (drizzel orm atua prisma coba sesuaikan denga nrekomendasi kamu) pastikan query jga ada buildernya agar memudahkan ai untuk
eksekusi kode secara mudah buatkan dokumentasi lengkap agar memudahkan
